## What I Learned

Throughout this course I learned the fundamentals of cloud computing using Amazon Web Services (AWS). I gained hands-on experience working with EC2 instances, S3 storage buckets, Amazon RDS databases, VPC networking, Elastic Load Balancing, Auto Scaling, and AWS Lambda. These labs helped me understand how organizations deploy applications in the cloud while maintaining security, availability, and scalability. I also learned the AWS Shared Responsibility Model and how cloud resources can be managed efficiently using AWS management tools.

## What Challenged Me

One of the biggest challenges was understanding how different AWS services work together. At first, concepts such as VPC networking, security groups, subnets, and load balancers seemed confusing. During several guided practice labs I had to carefully follow the instructions and troubleshoot configuration issues. By repeating the exercises and reviewing AWS documentation, I gradually became more comfortable with the cloud environment. If I completed the labs again, I would spend more time planning the architecture before beginning deployment.
